## 📖 Sobre o Projeto

O **Travelgram** é uma página web que simula a interface de uma rede social de fotos, inspirada no design do Instagram.

Este projeto foi desenvolvido com o objetivo de **praticar HTML e CSS**, focando em:

* 📸 Estrutura de feed de fotos
* 👤 Seção de perfil de usuário
* 🎨 Layout moderno e organizado
* 📱 Estrutura semelhante a redes sociais populares
